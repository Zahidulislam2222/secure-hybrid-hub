# Hybrid Dossier

Use `hub.py dossier show` for local approved facts and `dossier export-safe` only
for explicitly selected projection sections. Submit typed proposals for changes;
do not directly edit approved facts. Architecture, security, compliance, cloud,
environment, ownership, deployment, and exception changes require human
approval. Never place raw secrets, records, privileged material, or logs in a
dossier.
